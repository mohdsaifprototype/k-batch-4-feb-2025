CSS Pseudo Selector

A Pseudo selector is a special kind of selector that is used to style the state of an element. A pseudo selector is part of CSS only. and it starts with : or :: sign.

There are two types of pseudo selector. such as pseudo class and pseudo element

  1. Pseudo Class

    It is declared using single colon sign. i.e., :pseudo-selector

  2. Pseudo Element

    It is declared using single colon sign. i.e., ::pseudo-selector
