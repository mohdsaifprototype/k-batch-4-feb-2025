  • Introduction

    What is CSS?

    CSS is a style sheet that controls the appearance of a document.

    What is the full form of CSS?

    Cascading Style Sheets

  • Methods of CSS

    There are 3 methods to apply CSS.

    1. Inline
      What is Inline CSS?

      Inline CSS is used to apply styles to a single element.

      Example: <p style="color: red;">Hello World!</p>

      Click here to see the example
    2. Internal
      What is Internal CSS?

      Internal CSS is used to apply styles to a web page.

      Example: <style> p { color: red; } </style>

      Click here to see the example
    3. External
      What is External CSS?

      External CSS is used to apply styles to a website.

      Example: <link rel="stylesheet" href="styles.css">

      Click here to see the example
      Note: Internal and External CSS syntax is must be mentioned within the head element.
